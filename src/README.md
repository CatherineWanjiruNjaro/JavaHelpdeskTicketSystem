# Java IT Helpdesk Ticket System

This is a simple Java desktop application that allows IT support staff to manage helpdesk tickets.

## Features

- Create a new support ticket
- View all tickets
- Update ticket status
- Search for a ticket by ID
- Uses Java JOptionPane for user interaction

## Technologies Used

- Java
- Object-Oriented Programming
- ArrayList
- JOptionPane GUI

## How to Run

1. Open the project in VS Code.
2. Compile the program:

```bash
javac src/HelpdeskTicketSystem.java
